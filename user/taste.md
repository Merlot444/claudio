# 我的音乐品味关键词

这份文件只提供选曲倾向，不是硬性规则。用户明确点歌或明确指定风格时，以用户当次输入优先。默认华语中文歌占 30% 权重，其他 70% 权重。

## 高频关键词
- 律动感 / Groove
- 重低音 / Bass-heavy
- 节奏驱动
- 氛围感 / Atmospheric
- 街头感 / Street
- 能量爆发
- 采样艺术
- 合成器 / Synth
- 808 鼓机
- 自动调音 / Auto-tune
- 循环节拍 / Loop-based
- 电子音色
- 迷幻 / Psychedelic
- 工业感 / Industrial
- 故障美学 / Glitch
- 回响 / Reverb-drenched
- 脉冲感 / Pulse
- 极简节奏 / Minimal beat

## 轻微负向关键词
- 过度甜腻的流行情歌
- 无节奏起伏的纯钢琴
- 民谣叙事过长
- 过于感伤缓慢
- 无铺垫强炸（除非用户主动要求高能）
- 纯自然白噪音无音乐性

## 播报语气关键词
- short
- cool
- laid-back
- appreciative of the beat
- no over-explaining
- no therapy tone
- no cheesy motivational speech
