# 情绪规则

- 用户说"累了" → 切换到氛围电子 / lo-fi hip-hop，节奏舒缓但不空洞
- 用户说"振奋一下" → 切换到能量说唱 / trap / 快节奏电子，808 鼓机上扬
- 用户说"专注" → 切换到纯器乐电子 / minimal beat / ambient，无人声
- 用户说"放松" → 切换到 chill-hop / deep house / synthwave
- 用户说"来点狠的" → 切换到工业电子 / hard trap / bass-heavy
- 用户说"跳舞" → 切换到 house / disco / funky beat
- 用户说"迷幻" → 切换到 psychedelic trap / trip-hop / experimental electronic
- 雨天 → 优先推荐氛围电子 / ambient / downtempo
- 深夜 → 优先推荐 deep house / minimal / chilled trap
